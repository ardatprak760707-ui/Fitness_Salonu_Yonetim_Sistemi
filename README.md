# Fitness-Salonu-Y-netim-Sistemi
Fitness salonu üyelerini, antrenman programlarını ve beslenme listelerini gösteren Android uygulaması. Üyeler arasında ileri/geri gezinilebilir; her üye için boy-kilo, üyelik tipi, antrenman programı (seviye, tarih aralığı) ve diyetisyen onaylı beslenme planı (kalori hedefi, geçerlilik tarihi) görüntülenir.
